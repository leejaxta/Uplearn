# Uplearn_Backend

A simple backend for the Uplearn platform.  
This project uses **JSON Server** to serve data and hosts the `uploads` folder **statically** to allow access to uploaded files.

---

## Getting Started

Follow these steps to get the backend running locally.

### Requirements

- Node.js >= 18
- npm >= 9

### Installation

Clone the repository:

```bash
git clone https://git.bajratechnologies.com/j-25-06/j-25-06-angular/leeja/uplearn_backend.git
cd uplearn_backend

npm install
```

## Running the Server

Start the backend:

```bash
node server.js
```
