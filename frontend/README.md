# Uplearn_Frontend

The frontend of the **Uplearn platform**, built with **Angular**.  
This project provides the user interface for learners and instructors, including course browsing, enrollment, quizzes, and progress tracking.

---

## Getting Started

Follow these steps to get the frontend running locally.

### Requirements

- Node.js >= 18
- npm >= 9
- Angular CLI (`npm install -g @angular/cli`)

---

## Installation

Clone the repository and install dependencies:

```bash
git clone https://git.bajratechnologies.com/j-25-06/j-25-06-angular/leeja/uplearn.git
cd uplearn
npm install
```

## Running the Project

Start the Angular development server:

```bash
ng serve
```
